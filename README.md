该项目包含react、react-router-dom、redux、react-mobx、mobx-state-tree、hooks基本用法示例,后期将逐步更新
**项目执行命令**
1. 安装依赖
    `yarn install`
2. 启动项目
    `yarn start`
3. 构建项目
   `yarn build`  
4. 单元测试   
 `yarn test`
5. 代码检测
  `yarn lint`
  使用eslint 和 stylelint 同时对js 和 css 文件进行检测
6. 代码修复
   `yarn fix`
   同时修复js和css文件
7. 代码提交
 `yarn git`
 提交前进行代码检测
